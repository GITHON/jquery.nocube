## jquery插件nocube
nocube 取自nosql 表示not only cube 是一个可以将多个元素变为一个可以可以用手拖动的正多边形立方体的jquery插件

### 使用方法
```
引用js与css文件

基本文档结构
<div id="**">
    <div class="nocube-stage">
        <div class="nocube-wrapper">
            <div></div>
            <div></div>
            <div></div>
            ...
        </div>
    </div>
</div>

$('#**').nocube()
```
